# Project Statement

## 1. Problem Statement
Managing student records manually can be time-consuming and prone to errors. In many cases, maintaining data such as student names, roll numbers, and marks in notebooks or spreadsheets becomes inefficient, especially when updates or searches are required. The goal of this project is to develop a simple command-line based Student Management System that allows users to efficiently store, manage, and update student information.

## 2. Scope of the Project
The project involves:
- Creating a menu-driven command-line application.
- Implementing functionalities to add, view, search, update, and delete student records.
- Storing data using a JSON file for persistence.
- Ensuring the program runs through a terminal environment.
- Organizing the project with proper documentation and GitHub repository.

## 3. Target Users
This project is designed for:
- Students learning basic programming and file handling concepts.
- Teachers or individuals who need a simple system to manage student data.
- Beginners who want to understand how real-world applications are built using Python.

## 4. High-Level Features
- **Add Student**: Insert new student details such as name, roll number, and marks.
- **View Students**: Display all stored student records.
- **Search Student**: Find a student using roll number.
- **Update Student**: Modify existing student details.
- **Delete Student**: Remove student records from the system.
- **File Storage**: Data is stored in a JSON file for permanent access.
- **Command-Line Interface**: Fully executable through terminal without GUI.
- **GitHub Repository**: Contains source code, README, and documentation.
