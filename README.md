# Student Management System (CLI)

## Description
This is a command-line based Student Management System built using Python. It allows users to manage student records.

## Features
- Add student
- View students
- Search student
- Update student
- Delete student

## Requirements
- Python 3.x

## How to Run

1. Clone the repository:
   git clone https://github.com/nandini25bec10042-tech/Student-Management-System

2. Go to folder:
   cd Student-Management-System

3. Run:
   python main.py

## Output
The program runs in terminal and allows user interaction.
